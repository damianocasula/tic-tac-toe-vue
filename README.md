# tic-tac-toe-vue

## Tic Tac Toe in Vue!
A project for [Start2impact](https://start2impact.it).

## Project setup

### Install dependencies
```
npm install
```

### Hot-reload server for development
```
npm run serve
```

### Deploy to GitHub

1. Use deploy-example.sh as a reference to create a deployment script named `deploy.sh`.
2. `chmod +x deploy.sh`
3. `./deploy.sh`
