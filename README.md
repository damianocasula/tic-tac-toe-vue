# Tic Tac Toe in Vue!

## Project setup

### Requirements

- NodeJS
- NPM
- Vue CLI

### Install dependencies
```
npm install
```

### Hot-reload server for development
```
npm run serve
```

### Deploy to GitHub

1. Use `deploy-example.sh` as a reference to create a deployment script named `deploy.sh`.
2. `chmod +x deploy.sh`
3. `./deploy.sh`
